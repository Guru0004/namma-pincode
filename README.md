# Namma Pincode 📍

Namma Pincode (Bangalore Pincode Explorer) is a full-stack web application that allows users to search for area names by pincode and vice versa within Bangalore. 

## 🛠️ Technologies Used

### Frontend (Client)
- **Framework:** React 19
- **Build Tool:** Vite
- **Styling:** CSS / TailwindCSS (depending on implementation specifics)

### Backend (Server)
- **Runtime:** Node.js
- **Framework:** Express.js
- **Middleware:** CORS

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### 1. Clone the repository
\`\`\`bash
git clone <repository-url>
cd "namma pincode"
\`\`\`

### 2. Setup the Server
Open a terminal and navigate to the `server` directory:
\`\`\`bash
cd server
npm install
node index.js
\`\`\`
The server should start running, typically on `http://localhost:3000` (or whichever port is defined in your backend).

### 3. Setup the Client
Open a new terminal window and navigate to the `client` directory:
\`\`\`bash
cd client
npm install
npm run dev
\`\`\`
This will start the Vite development server, usually on `http://localhost:5173`. Open this URL in your web browser to view the application.
